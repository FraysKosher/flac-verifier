How it works

Three steps to the truth

Drop a folder and walk away. FLAC Verifier handles everything else.

01

📁

Select a folder

Drag and drop any folder containing FLAC files, or browse to it. Single files work too.

02

📊

Full spectral analysis

Every file is processed in its entirety — spectral ceiling, dynamic range, bit-depth grid, clipping detection, and a generated spectrogram image.

03

✅

Automatic verdict

Each track receives a scored verdict: Lossless Genuine, Probably Lossless, Doubtful, or Probable Upscale — with a full explanation of every metric.

Features

Everything you need

Built for audiophiles, archivists, and anyone who cares about what's actually in their music collection.

🖼️

Spectrograms always included

Every analysis generates a full-track spectrogram PNG. View inline or open externally. No option to skip.

⚡

Batch analysis

Analyze an entire album folder in one click. Results stream in card by card as each file finishes.

🏷️

Four-tier verdicts

Scored from 0–100% with a clear verdict label:

GENUINE PROBABLE DOUBTFUL UPSCALE
🔔

Desktop notifications

Get a Windows system notification when analysis completes — useful for large albums running in the background.

🔍

Filter & history

Filter results by verdict with one click. Every analysis is saved to a cumulative CSV history log you can sort and search.

📄

PDF & CSV reports

Export a formatted PDF report and a machine-readable CSV alongside each analyzed folder.

Installation

Get started in minutes

A pre-built Windows installer is available. To run from source, you'll need Python 3.10+ and the dependencies below.

1

Clone the repository

Or download the latest release if you only want the installer.

2

Install Python dependencies

NumPy, SciPy, Mutagen, Matplotlib, fpdf2, and Pillow.

3

Launch the app

The Tauri desktop shell starts automatically and loads the UI.

# Clone
$ git clone https://github.com/FraysKosher/flac-verifier.git
$ cd flac-verifier

# Python dependencies
$ pip install -r requirements.txt

# Launch (dev mode)
$ cd flac-verifier
$ npm install
$ npm run tauri dev
